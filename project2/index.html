Part 1 — Image Homography Estimation

Objective

Estimate a planar projective transform (homography) to rectify or map between two views of the same plane. Demonstrate correct rectification and warping on real images.

Method (DLT + Normalization)

  • Collect ≥4 point correspondences between source and target planes.
  • Build the linear system \(Ah=0\) and solve via SVD for \(h\) (last singular vector).
  • Use coordinate normalization for numerical stability; denormalize at the end.
  • Apply inverse mapping with bilinear interpolation to avoid holes.

Implementation Notes

  • Output bounds computed from warped corner extents to size the canvas.
  • Masking & optional feathered blending to handle seams at boundaries.
  • Assumes a planar scene region; perspective camera model.

Results (Part 1)

  • Rectification successfully removes perspective distortion on planar targets.
  • Stable solutions with normalized DLT and RANSAC (if outliers present).
  • Clean warps with inverse mapping + bilinear interpolation.
